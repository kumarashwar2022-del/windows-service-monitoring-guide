# windows-service-monitoring-guide
Developed a Windows Service &amp; Process Monitoring Agent using Python to monitor running processes and Windows services. Implemented parent-child process analysis, startup service auditing, whitelist/blacklist-based process detection, logging, and automated report generation to identify suspicious or unauthorized system activities.
